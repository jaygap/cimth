# CIMTH

Acronym of "**C** **I**mage manipulator with **M**ulti-**T**hreading and **H**ardware acceleration" (pronounced 'symth').

## Why?

- It serves as a rewrite of a project I began in Java last year (https://github.com/jaygap/basic-image-manipulation).
- I want to learn more about C
- I want to learn about multithreading
- Why not

## Planned Features

- Pixel Sorting
- Edge Detection
    - Horizontal
    - Vertical
    - Sobel operator
    - Canny
    - Kovalevsky
- Blurring
    - Box
    - Gaussian
- Making an image greyscale
- Making a mask of an image
- Image resizing (up and down)
    - Nearest-neighbour
    - Box Sampling
    - Bilinear
    - Bicubic
- Image cropping
